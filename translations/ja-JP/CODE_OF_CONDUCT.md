# Contributor Covenant 行動規範

## 私たちの誓約

私たちはメンバー、コントリビューター、リーダーとして、年齢、体格、外見上または外見からは分からない障害、民族、性別特性、性自認および性表現、経験の程度、教育、社会経済的地位、国籍、容姿、人種、宗教、性的アイデンティティ、性的指向にかかわらず、すべての人がハラスメントを受けることなくコミュニティに参加できるようにすることを誓います。

私たちは、開かれた、歓迎的で、多様性と包摂性があり、健全なコミュニティに貢献する方法で行動し、交流することを誓います。

## 私たちの基準

前向きな環境づくりに貢献する行動の例は次のとおりです。

- 他者に共感と思いやりを示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを行い、それを真摯に受け入れること
- 責任を受け止め、自らの過ちによって影響を受けた人に謝罪し、その経験から学ぶこと
- 個人の利益だけでなく、コミュニティ全体にとって最善となることを重視すること

容認できない行動の例は次のとおりです。

- 性的な言葉や画像の使用、および望まれない性的な注目や誘いかけ
- 荒らし行為、侮辱的または軽蔑的なコメント、個人または政治に対する攻撃
- 公開または非公開の場でのハラスメント
- 明示的な許可なく、住所やメールアドレスなど他者の個人情報を公開すること
- その他、専門的な環境において不適切であると合理的に判断される行為

## 執行責任

コミュニティリーダーは、容認される行動の基準を明確にして執行する責任を負い、不適切、脅迫的、攻撃的、または有害と判断した行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

本行動規範はすべてのコミュニティ空間に適用され、個人が公の場でコミュニティを公式に代表する場合にも適用されます。

## 執行

虐待、ハラスメント、その他の容認できない行動は、リポジトリのメンテナーに非公開で連絡して報告できます。すべての申し立ては、迅速かつ公正に確認および調査されます。
